# 2.Multi-Pipe
>Shell_Pipe 구현 
* PIpe란?</br> 
* 실행된 프로그램이나 명령어 등의 결과를 다른 프로그램으로 넘겨 줄 때 사용</br>
* Pipe는 command1 | command2과 같이 입력되었을때 command1의 출력을 command2의 입력으로 하여 연결</br>
